# Changelog

All notable changes to CodexDeck are documented here.

This project follows semantic versioning where practical.

## [0.1.0] - 2026-05-15

### Added

- Browser-first control page for local Codex CLI sessions.
- PTY-backed managed sessions with Socket.IO streaming.
- Recent thread discovery from Codex state.
- Read-only JSONL watch mode for external Codex sessions.
- Browser-controlled resume mode for existing threads.
- Structured reply rendering with message timestamps and expandable command output.
- Mobile-friendly session picker and input controls.
- Token auth, account + TOTP auth, and Tailscale helper scripts.
- CI and tag-driven GitHub Release workflow.
