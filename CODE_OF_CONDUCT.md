# Code of Conduct

CodexDeck follows the Contributor Covenant Code of Conduct, version 2.1.

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful and direct.
- Giving and accepting constructive feedback.
- Focusing criticism on the work and its impact.
- Being mindful that this project touches local shells and private code.

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language.
- Publishing private information, credentials, logs, or tokens.
- Deliberately introducing unsafe defaults or misleading security guidance.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior is harmful to the project or its users.

Instances can be reported through GitHub issues or security advisories when privacy is needed.
